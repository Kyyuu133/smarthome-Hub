 Smart Home Hub (FastAPI + SQLite)

Ein einfaches Smart-Home-Backend mit Geräteverwaltung, Benutzerverwaltung und Tages-Simulation.

Das Projekt simuliert ein Smart-Home-System mit:

Lampen (inkl. Helligkeit)
Weckern
Thermostat-Logik
Tagesverlauf (Day Emulator)
REST API mit FastAPI
SQLite Datenbank

 Features

Geräte in Räumen verwalten
Benutzerverwaltung mit Sessions
Tages-Simulation mit automatischer Geräte-Steuerung
Temperatur-abhängige Steuerung (Thermostat Logik)
SQLite als persistente Datenbank
REST API mit FastAPI

 Verwendete Technologien

Python 3.x
FastAPI
SQLite
Starlette Sessions
